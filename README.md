🐾 Pet Health App

A mobile application developed using Kotlin that helps pet owners manage their pets’ health efficiently. The app integrates AI-powered diagnosis, clinic navigation, and scheduling features to provide a complete pet healthcare solution.

📱 Features
🗺️ Nearby Pet Clinic Navigation

Locate nearby veterinary clinics using Google Maps API

Real-time navigation and distance tracking

Easy access to clinic information

📅 Pet Appointment Scheduling

Book and manage vet appointments

Schedule medicine Consuming

Reminder functionality for appointments

🩺 Pet Health Diagnosis (AI-Based)

Upload pet images or input symptoms

AI analyzes possible skin diseases or conditions

Provides preliminary health insights (non-medical advice)

🤖 AI Chatbot Assistant

Interactive chatbot powered by OpenAI API

Answer pet health-related questions

Provide guidance and basic recommendations
